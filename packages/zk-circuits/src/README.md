# ZK Circuits

Circom circuits for zero-knowledge proof generation. Phase 3 implementation.

## Planned Circuits
- Identity verification (eye scan → ZK proof → on-chain match)
- Selective disclosure (prove facts about records without revealing them)
- Biometric match (prove eye matches enrolled identity without revealing iris template)

## Tools
- Circom compiler
- snarkjs for proof generation/verification
- Groth16 proving system
